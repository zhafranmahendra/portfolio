QR Code Door Lock Project

Project Overview

This is a project for a microprocessor system course on campus with the title of QR Code Door Lock System Development with Android Application. The hardware used includes ESP8266 Microcontroller, Bluetooth Module, Solenoid Door Lock, Buzzer, Button, Relay, LED. The main process of this project uses C programming language. This device will be connected by bluetooth from the user's mobile phone to communicate. If the scanned QR Code is valid, the door will be opened.

Android Application

At first, make sure the application bluetooth on the cellphone is turned on. This application allows the user to see the bluetooth connections that are around. Users are required to select a bluetooth device with the code HC-05. After connected, the user can scan the qr code according to the user's identity by clicking scan button. This application development process uses the MIT app inventor platform.

Desktop Application

In addition to android application features, this system has desktop-based application features. This application is used as feedback to the user regarding the successful access of the door lock. User identity will be displayed in the form of name and department.
