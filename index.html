1. Web-based Dashboard For Monitoring Client's Assets

My main responsibility in my current work is to create a web-based dashboard for monitoring clients’ assets such as temperature monitoring, equipment status monitoring, etc. Our clients are usually coming from manufacturing companies who want to improve their resource efficiency and monitor their assets easily in the era of digital transformation. I handled the front-end and the back-end service for each project. The pictures beside represent 2 dashboard development projects that have been carried out including: a Web-based Dashboard for projector usage monitoring in Panasonic Singapore office (Smart Projector Project) and Web-based Dashboard for asset monitoring at Sri-Trang's Thailand Gloves Factory. The website development platform used is PTC Thingworx. This platform uses javascript to build the back end service.

2. Employee Attendance System Based on Face Recognition

This is my bachelor thesis project with the title of designing a facial recognition-based employee attendance system. The problem that must be solved is the low integrity of the attendance system because it is still easy to falsify the attendance data of members of the organization. In addition, the issue of ease of use and the COVID-19 virus which requires a lot of less-contact activities in an effort to suppress the spread of the virus also needs to be considered. I am responsible for creating machine learning models, pre-processing datasets, and integrating programs into databases. The programming language used is Python and the data is stored in MySQL Database.

3. Face-mask Detection

This is one of projects I worked on during my internship at PT Tower Bersama Group, entitled the development of a monitoring system for the use of employee masks at PT Tower Bersama Group's office.
