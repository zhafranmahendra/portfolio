Digital Clock with HMI Project

Project Overview

This is a project for a microprocessor system course with the title of Digital Clock Development with HMI (Human Machine Interface). This digital clock has various features such as setting hours and minutes, setting alarms via buttons, and setting alarms via HMI. The programming language used for the main process is C, while the HMI is made using visual basic.

Human Machine Interface (HMI)

On this HMI, hours, minutes, seconds are displayed according to the actual conditions of the digital clock. In this HMI there is also a feature to set an alarm if the user feels lazy to set an alarm using the buttons on the device.
